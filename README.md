🕷️ Catálogo de Trajes de Spider‑Man
Este proyecto es una página web creada con HTML y CSS que presenta una colección de los trajes más icónicos de Spider‑Man. Incluye imágenes, descripciones detalladas y una estructura organizada en secciones para explorar la evolución del héroe a través de su vestuario.
🎯 Objetivo del proyecto:
* El propósito de este proyecto es practicar y demostrar conocimientos fundamentales de desarrollo web:
* Estructura semántica en HTML
* Organización de contenido en listas y secciones
* Uso de CSS para tipografías, colores y estilos
* Manejo de rutas relativas para imágenes
* Creación de una página navegable y visualmente clara

🧩 Contenido principal
La página incluye:

--> Una introducción a la historia del traje de Spider‑Man
--> Secciones dedicadas a la evolución del diseño
--> Una reflexión sobre la relación entre identidad y vestimenta
--> Una galería con los trajes más representativos, cada uno con:
--> Imagen
--> Nombre del traje
--> Descripción detallada
🛠️ Tecnologías utilizadas
1) HTML5
2) CSS3
3) Imágenes en formato JPEG y WEBP
